1) World Wide Web General

1.1 What are the most essential technologies that made the World Wide Web such a success?

A: UDI( universal document identifier ), URL( uniform resource locator ), URI( uniform resource identifier ), HTML( HyperText Markup Language ), HTTP( HyperText Transfer Protocol )

1.2 How are resources (html pages, images, etc) addressed in the web?

A: With an URL

1.3 What are the general components of a url?

A: A scheme, a host, a path and a query string

2) Web Server

2.1 What is a Web Server?

A: Web server refers to server software, or hardware dedicated to running said software, that can serve contents to the World Wide Web

2.2 What does a Web Server?

A: Store, process and deliver web pages to clients.

2.3 How can content be stored on a web server?

A: With a full implementation of HTTP,which includes ways of receiving content from clients

2.4 How is content in webpages described?

A: A real file on the server's secondary storage

3) Web Client

3.1 What is a web client?

A: A web browser (commonly referred to as a browser) is a software application for accessing information on the World Wide Web. Each individual web page, image, and video is identified by a distinct URL, enabling browsers to retrieve and display them on the user's device.

3.2 What does a web client?

A: The purpose of a web browser is to fetch information resources and display them on a user's device.

3.3 Web Client and Web Browser is the same?

A: Yes, it is.
